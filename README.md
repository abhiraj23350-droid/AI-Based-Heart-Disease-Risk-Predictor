## 🎯 Project Overview

An AI-powered heart disease risk predictor using **Random Forest Classifier** trained on **918 patients** with **87.5% accuracy**.

This Coder workspace gives you a **one-click Jupyter environment** to run the full ML pipeline.

---

## 🚀 How to Use (Coder)

1. Go to **Coder Dashboard** → Templates → Import this template
2. Click **"Create Workspace"**
3. Fill in the parameters:
   - **Project Name** — give your workspace a name
   - **Python Version** — 3.11 recommended
   - **Enable GPU** — optional
4. Wait ~2 minutes for setup
5. Click **"❤️ Heart Disease AI — Jupyter"** button
6. Run all cells in `heart_disease_ai.ipynb`!

## 🤖 ML Model Details

| Property | Value |
|----------|-------|
| Algorithm | Random Forest Classifier |
| Dataset | 918 patients (Byte2Beat) |
| Features | 16 medical features |
| Train/Test Split | 80% / 20% |
| Accuracy | **87.5%** |
| F1-Score | **0.87** |
| Explainability | SHAP Analysis |

---

## 📊 Top Risk Factors

1. **ST Slope** (ECG pattern) — most important
2. **Exercise Angina** — chest pain during exercise
3. **Oldpeak** — heart stress level
4. **Cholesterol** — blood cholesterol level
5. **Max Heart Rate** — maximum bpm achieved
